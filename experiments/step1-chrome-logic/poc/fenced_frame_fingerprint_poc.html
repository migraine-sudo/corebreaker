Fenced Frame Fingerprinting PoC

This page demonstrates that fingerprinting APIs are accessible from within a page context.

In a real scenario, this code would run inside a <fencedframe> served via Protected Audience API.

Since fenced frames require HTTPS and special ad auction setup, this PoC runs the same code in a regular page to demonstrate the APIs work.
