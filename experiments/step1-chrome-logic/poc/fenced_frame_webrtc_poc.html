Fenced Frame WebRTC P2P Leak PoC

This PoC demonstrates that fenced frames can create RTCPeerConnection and gather ICE candidates, leaking local IP addresses.

Instructions

  1. Open this page in Chrome with fenced frames enabled
  2. The fenced frame below should attempt to create RTCPeerConnection
  3. If vulnerable: ICE candidates with local IPs will appear
  4. If mitigated: RTCPeerConnection creation should throw DOMException
